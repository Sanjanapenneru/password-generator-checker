## Password Generator & Strength Checker (Python)

This is a beginner-friendly Python project that allows users to:
- Generate strong random passwords
- Check the strength of a given password

The project runs in the terminal and is menu-based.

##  Features

- Generate secure passwords using letters, numbers, and symbols
- Check password strength (Weak / Medium / Strong)
- Simple and easy to use
- Beginner-friendly Python code
